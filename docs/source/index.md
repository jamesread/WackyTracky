# WackyTracky

WackyTracky is a minimal, keyboard-friendly task and list tracker.

## Overview

- **Frontend:** Vue + Vite, with Vue Router and picocrank. Optimized for power users and GTD-style workflows.
- **Backend:** Go service (see `service/`). Supports a todotxt backend among others.
- **Protocol:** ConnectRPC/gRPC with Protocol Buffers (see `protocol/`).
